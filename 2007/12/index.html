  • Growl for Entourage (via Applescript) December 04, 2007 So I went looking for a way to connect Microsoft Entourage (yeah, I'm in Exchange land now) with Growl for notifications. I've put it in [my public subversion repo](http://svn.geekdaily.org/public/applescript), so if you need it either ctrl-click and download or use svn to nab it.
