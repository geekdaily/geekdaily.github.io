Installing Linux Mint on Lenovo G50-45

I’m dual-booting just to make it easy to revert to Windows if and when.

Learned that the SD card system is PCI-Express based, so no booting from that. Pooh. Trip to Radio Shack to buy writable DVD material. Felt like 1999 again.

Debian * Set up your Lenovo * Shrink the disk partition * http://www.everydaylinuxuser.com/2015/11/how-to-shrink-windows-10-to-make-space.html * Decide local or network install. I chose network. * Get the right install image to boot from https://www.debian.org/CD/netinst/ * I chose amd64 * Make a bootable SD card * Mac: https://www.youtube.com/watch?v=CLVI0Q5hlz0 (verbose, but if you’re not a sysadmin) * Or textually:

Linux Mint

  1. Set up Windows
  2. Shrink the disk partition
    • http://www.everydaylinuxuser.com/2015/11/how-to-shrink-windows-10-to-make-space.html
  3. Get the right install image to boot from http://linuxmint.com/download.php
    • I chose Cinnamon since it’s based on Gnome 3.x
  4. Make a DVD
  5. Boot the DVD and wait … and wait … while it auto logs in
  6. Make your own partitions: http://winaero.com/blog/how-to-partition-your-hard-drive-to-install-linux-mint/
    • 1 MB reserved for bios (grub) /sda8
    • 300 MB /boot /dev/sda9
    • 20480 MB / /dev/sda10
    • 20480 MB /altroot /dev/sda11
    • 32 MB swap at end of space /dev/sda12
    • all the rest /home /dev/sda13
  7. Install
  8. Update kernel
    • https://mintguide.org/system/561-install-linux-kernel-4-4-lts-on-linux-mint.html
    • Get kernel from http://kernel.ubuntu.com/~kernel-ppa/mainline/ (instrux: https://wiki.ubuntu.com/Kernel/MainlineBuilds)
    • dpkg -i *deb && update-grub && reboot
  9. Update wireless drivers
    • Relevant bug is https://bugs.launchpad.net/ubuntu/+source/linux/+bug/1436940
    • Related, different rev: https://bugs.launchpad.net/ubuntu/+source/linux-firmware/+bug/1520343
    • Forum post: https://forums.linuxmint.com/viewtopic.php?t=196363
    • https://wireless.wiki.kernel.org/en/users/drivers/ath10k
    • Directory you need is QCA6174; added in 3.20 kernel
  10. apt-get update && apt-get dist-upgrade -y
    • Fixes system time issue, though you can also do …
  11. Two-finger scrolling off by default
    • https://www.maketecheasier.com/enable-two-finger-scrolling-in-linux-mint/

System time * hwclock –set –utc –date=”YYYY-mm-dd HH:MM:SS” * change /etc/default/rcS setting: “UTC=yes”
